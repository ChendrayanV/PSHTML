# h5

## SYNOPSIS
Create a h5 title in an HTML document.

## SYNTAX

``` powershell
h5 [[-Content] <Object>] [[-Class] <String>] [[-Id] <String>] [[-Style] <String>] [[-Attributes] <Hashtable>]
 [<CommonParameters>]
```
